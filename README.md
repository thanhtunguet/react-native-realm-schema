# react-native-realm-schema
Well-defined schema syntax for React Native Realm
## Installation

```sh
npm install react-native-realm-schema
```

## Usage

```js
import { multiply } from "react-native-realm-schema";

// ...

const result = await multiply(3, 7);
```
